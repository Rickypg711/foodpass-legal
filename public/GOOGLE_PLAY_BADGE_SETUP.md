# Google Play Badge Setup

## Current Implementation

`download.html` ya usa el asset PNG local oficial:

```html
<img src="/google-play-badge.png" alt="Disponible en Google Play" class="badge-img">
```

## Refresh del Badge Oficial

1. Ve a https://play.google.com/intl/en_us/badges/
2. Descarga los assets oficiales
3. Reemplaza `public/google-play-badge.png` manteniendo el mismo nombre
